# 🎮 Number Guessing Game in C

A simple console-based Number Guessing Game written in C.

## 📌 Features
- Generates a random number between 1 and 100
- Gives hints (Higher/Lower)
- Counts the number of attempts
- Simple and interactive gameplay

## 🛠️ Technologies Used
- C Programming
- Standard C Libraries (`stdio.h`, `stdlib.h`, `time.h`)

## ▶️ How to Run

```bash
gcc main.c -o game
./game
```

## 📷 Sample Output

```
Welcome to the Number Guessing Game!

Please enter your guess (1-100): 45
Guess a larger number.

Please enter your guess (1-100): 72
Guess a smaller number.

Please enter your guess (1-100): 63
🎉 Got it! You guessed it in 3 attempts.
```
